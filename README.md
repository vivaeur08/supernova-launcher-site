# 🪐 SuperNova Launcher — Site Officiel

Site web officiel du launcher Minecraft SuperNova.

## 🌐 Site en ligne

**[https://supernova-launcher.vercel.app](https://supernova-launcher.vercel.app)**

## 🚀 Stack

- HTML5 / CSS3 / JavaScript vanilla
- Aucun framework
- Déployé sur Vercel

## 🎨 Design

- Glassmorphism noir & violet
- Animations CSS + Web Audio API
- Responsive (mobile/tablet/desktop)
- Particules animées en background
