# 🌟 Zenra Launcher — Site Officiel

Site web officiel du launcher Minecraft Zenra.

## 🌐 Site en ligne

**[https://zenra-launcher.vercel.app](https://zenra-launcher.vercel.app)**

## 🚀 Stack

- HTML5 / CSS3 / JavaScript vanilla
- Aucun framework
- Déployé sur Vercel

## 🎨 Design

- Glassmorphism vert moderne
- Animations CSS fluides
- Responsive (mobile/tablet/desktop)
- Particules animées en background
