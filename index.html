FAQ

Questions fréquentes

Tout ce que tu dois savoir avant de te lancer.

Est-ce que c'est légal ?

SuperNova Launcher est un launcher open source distribué gratuitement. Cependant, jouer à Minecraft sans avoir acheté le jeu officiel n'est techniquement pas légal selon les CGU de Mojang. Nous recommandons fortement d'acheter Minecraft si tu en as les moyens.

Ce launcher a été créé pour permettre à des amis de jouer ensemble dans un cadre privé entre amis.

Comment installer le launcher ?
  1. Télécharge le fichier .exe depuis ce site
  2. Double-clique pour lancer l'installateur
  3. Suis les étapes (1 clic suffit)
  4. Lance le launcher depuis le raccourci Bureau
  5. Connecte-toi avec Discord
  6. Choisis un modpack et joue !
Mes amis ont aussi besoin du launcher ?

Oui, tous les joueurs qui veulent rejoindre les serveurs doivent installer SuperNova Launcher et être membres du serveur Discord SuperNova.

C'est ce qui permet la synchronisation des skins et la sécurité anti-triche.

Ça marche sur Mac / Linux ?

Pour le moment, seul Windows 10/11 est supporté. Une version Mac et Linux est envisagée pour le futur.

Si tu es sur Mac/Linux, tu peux utiliser un launcher classique comme TLauncher en attendant.

Comment signaler un bug ?

Le launcher possède un bouton "Signaler un bug" dans les paramètres. Ton rapport est envoyé directement sur le Discord de l'équipe et on le traite rapidement.

Tu peux aussi nous contacter via Discord pour discuter des problèmes.

Le launcher est vraiment gratuit ?

100% gratuit. Pas d'abonnement, pas de pub, pas de DLC. Le projet est développé bénévolement pour le plaisir et pour permettre à mes amis de jouer ensemble.

💜 Aucun paiement ne sera jamais demandé.

Mes données sont-elles en sécurité ?

Nous récupérons uniquement :

  • Ton pseudo Discord
  • Ton ID Discord
  • Ton avatar Discord
  • La liste des serveurs Discord auxquels tu appartiens (pour vérifier que tu es bien dans le serveur SuperNova)

Aucune autre donnée n'est collectée. Pas de mot de passe, pas de mail, rien.
