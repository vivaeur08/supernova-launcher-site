FAQ

Questions fréquentes

Tout ce que tu dois savoir avant de te lancer.

Qu'est-ce que Zenra Launcher ?

Zenra est un launcher Minecraft moderne qui permet de jouer aux modpacks officiels du serveur Zenra, mais aussi d'explorer et installer n'importe quel modpack de Modrinth ou CurseForge.

Il intègre un catalogue complet de mods, resource packs, shaders et datapacks, le tout dans une interface premium.

Comment installer le launcher ?
  1. Télécharge le fichier .exe depuis ce site
  2. Double-clique pour lancer l'installateur
  3. Suis les étapes (1 clic suffit)
  4. Lance le launcher depuis le raccourci Bureau
  5. Connecte-toi avec Discord
  6. Rejoins le serveur Discord (obligatoire)
  7. Choisis un modpack et joue !
Discord est-il vraiment obligatoire ?

Oui, la connexion Discord est obligatoire pour utiliser Zenra. Cela permet de :

  • Garantir une communauté unie
  • Recevoir des notifications
  • Synchroniser ton pseudo (mode offline)
  • Sécuriser l'accès (vérif serveur Discord)

Il faut aussi être membre du serveur Discord Zenra pour utiliser le launcher.

Ai-je besoin d'un compte Minecraft Premium ?

Non, Zenra fonctionne en mode offline avec ton pseudo Discord. Tu peux jouer en solo ou sur les serveurs privés.

Pour rejoindre les serveurs publics (Hypipxel, etc.), tu peux connecter un compte Microsoft Premium et switcher facilement entre les deux modes.

Ça marche sur Mac / Linux ?

Pour le moment, seul Windows 10/11 est supporté. Une version Mac et Linux est envisagée pour le futur.

Le launcher se met à jour automatiquement ?

Oui ! Zenra vérifie automatiquement les mises à jour à chaque démarrage. Une notification s'affiche en haut de l'écran quand une nouvelle version est disponible.

Un seul clic pour télécharger, et un autre pour redémarrer. Simple et rapide.

Le launcher est vraiment gratuit ?

100% gratuit. Pas d'abonnement, pas de pub, pas de DLC. Le projet est développé bénévolement.

💚 Aucun paiement ne sera jamais demandé.

Mes données sont-elles en sécurité ?

Nous récupérons uniquement :

  • Ton pseudo Discord
  • Ton ID Discord
  • Ton avatar Discord
  • La liste des serveurs Discord (pour vérifier que tu es dans le serveur Zenra)
  • Ton pseudo Microsoft (si tu connectes ce mode)

Aucune autre donnée n'est collectée.
